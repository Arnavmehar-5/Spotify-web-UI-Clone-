# Spotify-web-UI-Clone-

🔗 **[Live Demo — Spotify-web-UI-Clone](https://arnavmehar-5.github.io/Spotify-web-UI-Clone-/)**

A Spotify web UI clone built using HTML and CSS. This project recreates the core layout and visual design of Spotify's desktop interface, including the sidebar, playlists, charts section, and music player UI. The layout uses Flexbox throughout, with a fixed bottom player bar and a scrollable main content area. Built for learning and practicing frontend development fundamentals.
